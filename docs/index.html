QuickBooks MCP Server

A secure, self-hosted Model Context Protocol (MCP) server for querying QuickBooks data using natural language inside Claude Desktop.

About

This application runs on your own infrastructure and connects directly to the QuickBooks API using your credentials. No data is sent to third-party servers.

Getting Started

See the GitHub repository for setup instructions, including Docker and Synology Container Manager deployment.

Requirements

You need a QuickBooks developer account with API credentials (Client ID, Client Secret, Refresh Token, and Company ID). Visit developer.intuit.com to set up your app.
